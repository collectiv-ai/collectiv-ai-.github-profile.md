## About CollectiVAI

CollectiVAI is an independent project exploring **democratic, human-centered AI**
with a strong European focus.

The core idea is to:

- use AI to help people **understand** policies and proposals,
- support **participation** by citizens, experts and institutions,
- and build **transparent infrastructure** for democratic decision-making.

---

## Key building blocks

- **CollectiVAI App** ‚Äì iOS / iPadOS / macOS client for:
  - AI-assisted explanations of policies and proposals  
  - civic use cases for cities, universities, NGOs and startups  

- **CollectiVAI Chain** ‚Äì planned Cosmos-based app-chain for:
  - transparent governance and voting  
  - proposals, votes and treasury movements on-chain  

- **CollectiVAI Router & Ecosystem**
  - multi-provider AI routing (OpenAI, local models, etc.)  
  - local macOS M2 Max lab for AI, security and blockchain (CollectiVAI control center)  

- **Public docs & website**
  - human-readable overview of the project  
  - business plan, roadmap and civic use cases
